# Address Book Application (C)
# Overview :
The Address Book Application is a console-based project developed using C programming. It is designed to store and manage contact details efficiently through a simple, menu-driven interface. The application supports creating, searching, updating, deleting, and displaying contact records, with data persistence achieved using file handling. This project demonstrates strong fundamentals of C, including structured programming, modular design, and basic data validation.

# Features :
- Add new contacts  
- Search contacts by name, mobile number, or email ID  
- Edit existing contact details  
- Delete contact records  
- Display all saved contacts  

 # Data Fields :
- Name  
- Mobile Number (with validation)  
- Email ID (with format verification)  

# Technical Concepts Used :
- Structures and arrays  
- Functions and pointers  
- Conditional statements and loops  
- File handling for persistent storage  
- String manipulation  
- Modular programming using multiple C files

gcc main.c addressbook.c -o addressbook

# Data Storage :
All contact details are stored in database.csv using file handling to ensure data persistence.
